# MyRecipeBook

A sample Dart command-line application.

## Project Structure

- `bin/` - Contains the application entrypoint (`myrecipebook.dart`)
- `lib/` - Contains library code
- `test/` - Contains unit tests

## Prerequisites

- Dart SDK 3.10.7 or higher

## Getting Started

### Install Dependencies

```bash
dart pub get
```

### Run the Application

```bash
dart run
```

### Run Tests

```bash
dart test
```

### Analyze Code

```bash
dart analyze
```

## Features

This is a basic Dart console application that demonstrates:
- Project structure with separation of concerns
- Unit testing with the `test` package
- Linting with the `lints` package
